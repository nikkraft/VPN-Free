## 📖 Инструкция по установке (Пример: HAPP)

Скопируйте необходимую ссылку из списка выше.

Откройте приложение HAPP (или др.) на устройстве.

Нажмите справа сверху на Три точки.

Выберите пункт Вставить из буфера обмена.

Готово.
Для стабильной работы регулярно обновляйте подписку используя кнопку Обновить

##  📖 Инструкция по установке (Пример: FClash)

Скопируйте необходимую ссылку из списка выше.

Откройте приложение FClash (или др.) на устройстве.

Выберите в "Режим исхождения".

Заходите во вкладку профили и нажимаете на плюсик и дальше URL и вставьте скопированую
ссылку.
Дальше зайдите во вкладку "Прокси" далее тест задерки.

И выберете сервер по пингу.

Готово.
